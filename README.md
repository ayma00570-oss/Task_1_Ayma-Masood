# CarePoint Hospital – Static Webpage Design

Project 1 | DecodeLabs Frontend Development Internship | Batch 2026

## About
A static hospital landing page built with HTML and CSS for a fictional hospital based in Lahore, Punjab.

## Sections
- Emergency strip, Header & Navigation
- Hero, Stats, About
- Departments, Doctors
- Appointment Form, Photo Gallery, Footer

## Technologies
- HTML5
- CSS3
- Google Fonts (Poppins)

## How to Run
Open `index.html` in any browser. Make sure `styles.css` is in the same folder. Internet connection required for images.

## Requirements Met
Semantic HTML, one h1, external CSS only, CSS Grid, Flexbox, alt text on images, no inline styles.# Task_1_Ayma-Masood
